# Diretório de Inteligência Artificial (.ai)

Este diretório centraliza todos os recursos e configurações relacionados a agentes de Inteligência Artificial (IA) utilizados neste projeto. O objetivo é fornecer um local padronizado para prompts, configurações, documentação e contextos, facilitando a colaboração e o desenvolvimento de soluções baseadas em IA.

## Estrutura de Pastas

- **/prompts**: Contém os prompts utilizados para interagir com os modelos de linguagem. Cada arquivo deve ser nomeado de forma descritiva, indicando sua finalidade.
- **/config**: Armazena arquivos de configuração para as ferramentas e serviços de IA.
- **/docs**: Guarda a documentação relevante sobre a arquitetura de IA, as decisões de design e as boas práticas adotadas no projeto.
- **/context**: Contém arquivos de contexto que fornecem informações específicas do projeto para os agentes de IA.
  - **/examples**: Subpasta para armazenar exemplos de dados, como arquivos CSV, JSON, etc., que podem ser usados como referência pelos agentes.

## Como Utilizar

Ao desenvolver novas funcionalidades com IA, utilize este diretório para armazenar os artefatos relevantes. Mantenha a estrutura organizada e documente as novas adições para garantir que todos os membros da equipe possam entender e colaborar de forma eficaz.
